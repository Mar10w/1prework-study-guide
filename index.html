HTML

  • the head element containes info about the web page
  • The body element represents the visible content shown to the user
  • Meta elements contain info about the page

CSS

  • A margin indicates how much space we want around the outside of an element
  • a padding indicates how much space we want around the inside of an element
  • width, height, and margin-left/ right determine the size and spacing of an image in this case the box the text is in
  • within the header/footer section, background color and color creates the blue banner with white text
  • within .card, border and box shadow create the colors and effect of the boxes surrounding each heading

GIT

  • git status: checks what branch you are currently on
  • git checkout -b branch-name: creates an new branch
  • git checkout origin main: returns to the main branch
  • ls: shows list of files in current directory
  • cd file name: moves into file directory
